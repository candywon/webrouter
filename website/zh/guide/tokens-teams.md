# 令牌与团队

## API 令牌

令牌是客户端用来认证 wr-proxy 的 API Key，每个令牌可以配置：

- **模型白名单** — 限制可用模型
- **配额** — Token 使用上限
- **速率限制** — 每分钟请求数
- **智能降级** — 出错时自动降级到经济型模型
- **脱敏** — 开启/关闭 PII 剥离

### 创建令牌

1. 进入 **令牌管理** → **+ 创建**
2. 填写名称并关联组织
3. 设置模型访问权限
4. 配置配额和速率限制
5. 开启智能降级（可选）

## 组织

组织树支持多层级结构：

```
技术部（公司）
├── 后端组
└── 前端组
AI 实验室（公司）
└── LLM Ops
产品组（公司）
```

每个令牌归属于一个组织，支持按组织的配额追踪和计费报表。

## 配额管理

配额按令牌设置：

- **总配额** — 最大 Token 消耗量
- **已用量** — 自动追踪
- **剩余量** — 自动计算

配额即将耗尽时可触发告警。

## 团队成员

每个令牌关联一个成员邮箱。管理员可在计费面板查看每人用量统计。
