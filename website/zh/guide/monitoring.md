# 健康监控

## 监控面板

提供所有 Provider 的实时概览：

- **状态** — 健康、警告、不可用、未知
- **延迟** — 毫秒响应时间
- **可用率** — 14 天历史曲线
- **错误率** — 失败请求百分比

## 健康检测

每 5 分钟检测一次：

```json
{
  "provider": "OpenAI",
  "status": "healthy",
  "latency_ms": 850,
  "checked_at": "2026-06-07T10:30:00Z"
}
```

## 历史数据

系统保留 14 天健康检测历史：

- **延迟趋势** — 折线图
- **状态分布** — 饼图
- **Provider 时间线** — 状态变更记录

## Provider 详情

点击查看详情：

- 当前状态和延迟
- 14 天健康历史
- 最新错误信息
- 渠道级指标（newapi/oneapi 类型）
