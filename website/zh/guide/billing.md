# 成本计费

## 概述

WebRouter 实时追踪 Token 用量和成本：

- 按模型的成本拆分
- 按令牌和组织的用量统计
- 30 天历史趋势
- 配额管理
- 预算告警

## 成本追踪

### 模型定价

在 **系统设置** → **模型定价** 中配置：

```json
{
  "model": "gpt-4o",
  "input_price": 2.50,
  "output_price": 10.00,
  "unit": "每百万 Token"
}
```

常见模型已预配置默认定价。

### 请求日志

每次 wr-proxy 处理的请求均记录：

- 模型、Provider、Token
- 输入/输出 Token 数
- 成本（根据定价自动计算）
- 延迟、状态、错误信息

保留 30 天请求历史。

## 配额

配额按 Token 级别设置：

| 字段 | 说明 |
|------|------|
| 总配额 | 允许的最大 Token 数 |
| 已用 | 自动统计 |
| 剩余 | 自动计算 |

配额耗尽后 Token 停止工作（除非开启智能降级）。

## 报表

CSV 格式导出：

- **Token 报表** — 按 Token 统计用量
- **Provider 报表** — 按 Provider 统计
- **模型报表** — 按模型统计
