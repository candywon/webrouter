# 企业知识库 — 捕获层设计

> 版本：v1.0 | 日期：2026-05-14

---

## 1. 捕获流程

```
handleProxy() 正常流程
       │
       ├── 1. 请求脱敏 (SanitizeRequest)
       ├── 2. 路由到Provider
       ├── 3. 获取响应
       ├── 4. 响应脱敏还原 (DesensitizeRestore)
       ├── 5. 回写给客户端
       │
       └── 6. [新增] 异步投递到知识捕获channel
                │
                ▼
           knowledgeCh (带缓冲1024)
                │
                ▼
           Knowledge Worker (Go协程)
                │
                ├── 信号筛选（规则）
                ├── 脱敏后存储到 raw 表
                └── 满足条件的标记为 pending（待LLM处理）
```

### 3.3 投递（在 handleProxy 响应回写后）

```go
// 在 handleProxy 的 defer 或响应回写之后
if token.KnowledgeCaptureEnabled {
    go func() {
        entry := KnowledgeEntry{
            RequestID:  requestID,
            TokenID:    token.ID,
            TokenName:  token.Name,
            Department: token.KnowledgeDepartment,
            Model:      modelName,
            Endpoint:   endpoint,
            Prompt:     sanitizedPrompt,
            Response:   sanitizedResponse,
            TurnCount:  extractTurnCount(requestBody),
            ClientIP:   clientIP,
            Timestamp:  time.Now(),
        }
        select {
        case knowledgeCh <- entry:
            // 投递成功
        default:
            // channel满，丢弃，不阻塞
            log.Printf("[knowledge] channel full, dropping request %s", requestID)
        }
    }()
}
```

关键设计：
- `select + default`：channel满则丢弃，绝不阻塞主流程
- 独立goroutine投递：不占用handleProxy的goroutine
- 脱敏后存储：复用现有desensitize模块的结果

### 3.4 Knowledge Worker

```go
func knowledgeWorker() {
    for entry := range knowledgeCh {
        processKnowledgeEntry(entry)
    }
}

func processKnowledgeEntry(entry KnowledgeEntry) {
    // 第1级：信号筛选
    if !shouldCapture(entry) {
        return  // 丢弃
    }

    // 写入raw表
    saveToKnowledgeRaw(entry)
}

// shouldCapture 第1级信号筛选
func shouldCapture(entry KnowledgeEntry) bool {
    // 1. 响应太短（<200字符），大概率是简单问答
    if len(entry.Response) < 200 {
        return false
    }

    // 2. 对话轮数太少（<3轮），大概率是简单查询
    if entry.TurnCount < 3 && len(entry.Response) < 500 {
        return false
    }

    // 3. 代码占比过高（>60%），大概率是编程
    if codeBlockRatio(entry.Prompt+entry.Response) > 0.6 {
        return false
    }

    // 4. 明确的翻译/改写意图
    if isTranslationOrRewrite(entry.Prompt) {
        return false
    }

    return true
}
```

## 4. 信号筛选规则详解

### 4.1 codeBlockRatio — 代码占比检测

```go
func codeBlockRatio(text string) float64 {
    // 统计 ```...``` 之间的字符数
    codeChars := 0
    inCodeBlock := false
    lines := strings.Split(text, "\n")

    for _, line := range lines {
        if strings.HasPrefix(strings.TrimSpace(line), "```") {
            inCodeBlock = !inCodeBlock
            continue
        }
        if inCodeBlock {
            codeChars += len(line)
        }
    }

    totalChars := len(text)
    if totalChars == 0 {
        return 0
    }
    return float64(codeChars) / float64(totalChars)
}
```

### 4.2 isTranslationOrRewrite — 翻译/改写意图检测

```go
var translationPatterns = []string{
    "翻译", "translate", "翻訳", "번역",
    "改写", "润色", "rewrite", "paraphrase",
    "换种说法", "rephrase",
}

func isTranslationOrRewrite(prompt string) bool {
    lower := strings.ToLower(prompt)
    for _, p := range translationPatterns {
        if strings.Contains(lower, p) {
            // 如果同时包含分析/计算等关键词，可能不只是翻译
            analysisKeywords := []string{"分析", "计算", "对比", "评估", "analyze", "compare"}
            for _, ak := range analysisKeywords {
                if strings.Contains(lower, ak) {
                    return false  // 包含分析意图，保留
                }
            }
            return true
        }
    }
    return false
}
```

### 4.3 extractTurnCount — 对话轮数提取

```go
func extractTurnCount(requestBody map[string]interface{}) int {
    messages, ok := requestBody["messages"].([]interface{})
    if !ok {
        return 1
    }
    return len(messages) / 2  // 每轮 = user + assistant
}
```

## 5. raw 表写入

func saveToKnowledgeRaw(entry KnowledgeEntry) {
    _, err := db.Exec(`
        INSERT INTO wr_knowledge_raw
        (request_id, token_id, token_name, model_name, prompt, response, turn_count, client_ip, status, created_at)
        VALUES (?, ?, ?, ?, ?, ?, ?, ?, 'pending', ?)`,
        entry.RequestID, entry.TokenID, entry.TokenName, entry.Model,
        entry.Prompt, entry.Response, entry.TurnCount, entry.ClientIP,
        entry.Timestamp,
    )
    if err != nil {
        log.Printf("[knowledge] failed to save raw entry: %v", err)
    }
}

## 7. 性能保障

| 指标 | 目标 | 保障措施 |
|------|------|----------|
| 捕获延迟 | <1ms | 异步channel，非阻塞select+default |
| 内存开销 | <50MB | channel缓冲1024 × 平均条目5KB ≈ 5MB |
| 代理P99延迟 | 无变化 | 捕获完全独立于代理主流程 |
| 丢包率 | <1% | channel缓冲1024足够应对突发，极端情况丢包优于阻塞 |

## 8. 开关控制

三级开关：

1. **全局开关**：wr-proxy 启动参数 `--knowledge-capture`（默认关闭）
2. **Token开关**：`wr_tokens.knowledge_capture_enabled`（默认0）
3. **部门字段**：`wr_tokens.knowledge_department`（必须填写才能开启捕获）

只有三级全部满足才进行捕获。
