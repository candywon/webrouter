# 09 - 数据库 (db.go)

## SQLite 路径
```
cfg.DBPath (默认 data/webrouter.db)
```

## 初始化

### InitDB(dbPath) 
- 打开SQLite (WAL模式, BusyTimeout 5s)
- 调用 migrate()

### migrate() 迁移
自动创建/修改以下表:
| 表名 | 关键字段 |
|------|---------|
| wr_providers | id, name, type, base_url, api_key, models, priority, weight, proxy_enabled, enabled, status, ... |
| wr_provider_ext | provider_id(PK), proxy_enabled, rate_limit_rpm, timeout_seconds, max_retries, cost_multiplier, priority, weight, supports_tools |
| wr_tokens | id, name, key, user_id, models, provider_ids, quota_total, quota_used, rate_limit_rpm, subnet_whitelist, smart_downgrade, enabled, expires_at |
| wr_request_logs | id, request_id, token_id, token_name, provider_id, provider_name, model_name, endpoint, input_tokens, output_tokens, status_code, latency_ms, cost_cents, is_stream, is_retry, error_message, error_type, client_ip, created_at |
| wr_channels | id, name, provider_id, api_key, models, enabled |
| wr_model_pricing | model(PK), input_price, output_price, is_default |

注意: 迁移用 ALTER TABLE ADD COLUMN + IF NOT EXISTS，不会删数据

## 核心函数

### LoadProviders() → []*Provider
- 查询 wr_providers + wr_provider_ext LEFT JOIN
- 解析Models为JSON数组
- 加载supports_tools

### LoadChannels(providers) → []*Provider
- 查询 wr_channels
- 每个Channel克隆父Provider，替换APIKey和Models
- Channel继承父Provider的SupportsTools
- 追加到providers列表

### UpdateProviderStatus(id, status, latencyMs, errMsg)
- UPDATE wr_providers SET status, last_latency_ms, last_error, last_check_at

### UpdateProviderQuota(id, used)
- UPDATE wr_providers SET quota_used = quota_used + used

### LoadTokenByKey(key) → *Token
- SELECT * FROM wr_tokens WHERE key = ? AND enabled = 1
- 返回nil=未找到

### DeductTokenQuota(tokenID, costCents) → bool
- UPDATE wr_tokens SET quota_used = quota_used + costCents WHERE id = ? AND (quota_total = 0 OR quota_used + costCents <= quota_total)
- 返回是否成功

### InsertRequestLog(log *RequestLog) → error
- INSERT INTO wr_request_logs (...)
- 时间格式: time.Now().UTC().Format("2006-01-02 15:04:05") (兼容Python SQLAlchemy)

### GetDailyUsage(days) → []map
- 按天聚合: date, total_requests, valid_requests, total_tokens, total_cost
- 有效请求: status_code < 400 AND is_retry = 0 AND COALESCE(error_message,'') = ''

### GetModelUsage(hours) → []map
- 按模型聚合: model_name, total_requests, valid_requests, input_tokens, output_tokens, avg_latency

## 时间格式陷阱
Go写入时间必须用 Format("2006-01-02 15:04:05") 而非直接 time.Time，
否则Python SQLAlchemy无法解析 "+0000 UTC" 后缀。
