💡 智能模型选择说明

            

              

                

                  
✨ model=auto / smart

                  

                    将 model 参数设置为 auto 或 smart，网关会分析请求内容的复杂度（输入长度、对话轮数、代码检测、工具调用、推理关键词、系统提示词六个维度），自动选择最合适的模型。
                  

                  

                    简单请求 → 经济型模型（快速省钱）

                    中等复杂度 → 标准模型（性价比）

                    复杂推理 → 高级模型（最强推理）
                  

                

                

                  
📝 使用示例

                  
curl GATEWAY_URL/v1/chat/completions \
  -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"auto","messages":[
    {"role":"user","content":"分析这段代码的问题"}
  ]}'

                  

                    智能降级也可在"令牌管理"中为成员开启，无需客户端修改 model 参数。
